# Security Policy

## Supported Versions

The following table describes which versions of JustTalk currently receive security updates and vulnerability fixes.

| Version  | Supported |
| -------- | --------- |
| 1.0.15   | ✅         |
| < 1.0.15 | ❌         |

As JustTalk is currently in its early stages, only the latest public release receives security updates. Users are encouraged to keep their application up to date.

---

## Reporting a Vulnerability

The security and privacy of JustTalk users are taken seriously.

If you discover a security vulnerability, **please do not open a public GitHub Issue**, as doing so could expose users to unnecessary risk before a fix is available.

Instead, use one of the following methods:

### Preferred Method

Use GitHub's **Private Vulnerability Reporting** feature available in this repository.

### Alternative Method

If Private Vulnerability Reporting is unavailable, [contact](mailto:codeadiksuu@gmail.com) the maintainer directly with a detailed report.

Please include:

* A clear description of the vulnerability
* Steps required to reproduce the issue
* Potential impact and severity
* Screenshots, logs, or proof-of-concept code (if applicable)

---

## What to Expect

After submitting a report:

1. An acknowledgment will typically be provided within **72 hours**.
2. The report will be reviewed and validated.
3. If confirmed, a fix will be developed and released as soon as possible.
4. Once the issue has been resolved, public disclosure may occur through release notes or repository documentation.

---

## Scope

Security reports are especially appreciated for issues involving:

* Authentication and authorization
* Account takeover scenarios
* Message encryption and decryption
* Data exposure or privacy leaks
* Firebase security rules
* Push notification security
* Media upload and storage security
* Sensitive information disclosure
* Remote code execution or privilege escalation

---

## Responsible Disclosure

Please provide maintainers with a reasonable amount of time to investigate and resolve reported vulnerabilities before publicly disclosing them.

Thank you for helping keep JustTalk secure.
